# Terraform Practice Lab

Terraformを何も知らない状態から実務運用まで学ぶための静的学習サイトです。GitHub Pagesで公開できるReact SPAとして作られています。

## Local development

```bash
npm install
npm run dev
```

## Checks

```bash
npm test
npm run build
```

## Source policy

教材内容はHashiCorp公式のTerraform Tutorials、Language、CLI、Style Guide、Modules、State、Test documentationを主な一次情報として構成します。ブラウザ演習は安全な疑似Terraformであり、実際のクラウドリソースやTerraform CLIは実行しません。
